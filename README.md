# Ntangl

**The Trust and Coordination Layer for Embodied AI**

Ntangl is an open standards initiative developing the architecture, protocols, reference implementations and governance required for secure collaboration between Embodied AI participants.

The objective is to enable Embodied AI systems from different manufacturers, owners and deployment environments to discover one another, establish trust, understand capabilities and operational context, collaborate towards shared goals, and operate safely without vendor-specific integration.

Ntangl is not a robot operating system.

Ntangl is not a fleet management system.

Ntangl is not a cloud platform.

Ntangl is a universal interoperability and collaboration layer.
